企業活動・二天一夜

4127 楷模第八屆交接｜溪頭米堤大飯店企業活動行程表

台中高鐵站集合出發，下坪自然教育園區、溪頭自然教育園區、孟宗竹林古戰場、鹿谷圍爐煮茶，享受森林浴與茶文化的二天一夜輕旅行。
日期 12/5（五）– 12/6（六）
集合地點 高鐵台中站 09:00
住宿 溪頭米堤大飯店

行程：高鐵台中站 09:00 集合出發 → 下坪自然教育園區 → 蝶舞花間無菜單料理 → 溪頭自然教育園區森林浴 → 溪頭米堤大飯店入住與晚宴。

Day 2｜12/6（六） 溪頭 → 鹿谷 → 台中高鐵站

行程：飯店早餐與自由活動 → 辦理退房 → 孟宗竹林古戰場、長源圳生態步道 → 紫林莊百年辦桌菜 → 鹿谷圍爐煮茶茶文化體驗 → 返回高鐵台中站。

行前必讀
  • 請準時於 09:00 抵達高鐵台中站集合，以免影響整體行程。
  • 建議攜帶：保暖衣物、輕便雨具、防滑好走的鞋子、個人常備藥品。

  • 飯店設施如泳池、三溫暖等，請遵守場館規範與開放時間。
